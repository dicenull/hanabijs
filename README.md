## 動作確認

パッケージのビルド

```sh
deno task build バージョン番号
```

ローカルパッケージのリンク

```sh
cd npm
npm link
cd ../example
npm link hanabijs
```

サンプルプロジェクトの実行

```sh
deno task dev
```
